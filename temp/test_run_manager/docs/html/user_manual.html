Build statuses page shows configured builds and their current statuses. The page refreshes automatically every thirty seconds. The content of the page depends on the "page view" mode. There are two view modes: list view and detailed view. To switch between these modes click "List view" or "Detailed view" links respectively. Differences between the modes are considered below.

1.1. List View Mode

This mode allows to see an overview of all build statuses on a single page. It is suitable for ongoing build statuses monitoring. When in the list view mode, the build statuses page shows a table with a list of builds and their current statuses (waiting, building, etc.) and a link to the page with the last build result, if any. The link is in red if the last build failed or in green if the last build was successful. A caption of the link tells the last build number and time.

1.2. Detailed View Mode

This mode allows monitoring of the detailed status of a selected build. In the detailed view the page it is split into two parts. On the left side there is a vertical navigation bar that allows selecting a particular build to view. The right side of the page contains detailed information about the current build status including:

  • Build name with the current build status.

  • Currently running build steps.

  • Links to the last build result including build logs, error quotes, and changes in the last build, build history, date and time of the last build.

  • Link to the last clean build, if any.

  • List of changes in the currently running build (or changes in the last build if the build is not running).

1.3. Disabling Page Refresh

To provide online infromation about build statuses the build status page refreshes automatically. To disable automatic refresh click on "Turn refresh OFF" link. The link can be found in the top right corner of the status page. Once the refresh is disabled, the links changes its caption to "Turn refresh ON". Clicking on this links will re-enable the automatic refresh of the builds status page.
